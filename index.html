Cookies

Cookies are a way for websites to store data for later visits. This can make users' experiences with sites much more convenient.
(e.g. "Remember me" when logging in).


However, convenience and security are often antithetical. While most websites claim to only use experience-enhancing cookies, they frequently store advertising data to track you across sites, with or without your consent.

Query Parameters

Query Parameters (a.k.a. Query Strings) are the list of variales past the ? in a URL:


http://example.com?parameter1=value1&parameter2=value2


URL parameters allow websites to store data when a link is shared. This can be useful for services designed to be shared, like DuckDuckGo


However, these query parameters can also be used to link you to the person whom you shared a website.

If you don't trust a URL's query parameters, delete them before sharing the URL.
